# Big Boy Sales Copilot 🚀

This is an AI-powered sales copilot that generates pre-call reports using company and prospect research.

## 🔧 How it works
1. Research Company
2. Research Prospect
3. Generate Pre-call Report

## 📊 Output Includes
- Company Overview
- Prospect Background
- Pain Points
- Talking Points
- Pitch Strategy

## 🧠 Prompt Used
You are a sales copilot.  

Follow this exact sequence:  

First, use the "Research Company" tool to gather company information.  
Then, use the "RESEARCH PROSPECT" tool to gather prospect information.  
Finally, use the "PRE-CALL REPORT GENERATOR" tool.  
When calling the PRE-CALL REPORT GENERATOR:  
Pass the output from "Research Company" as Company Summary  
Pass the output from "RESEARCH PROSPECT" as Prospect Summary  
Do not skip steps. Always follow this order.  

Before using prospect data, verify that the LinkedIn profile matches the company provided.
If there is a mismatch, mention it clearly and avoid making assumptions.

## 📌 Example Input
Company: Morningside AI  
Prospect: Ayush Sharma (Linkedin URL)

## 📌 Example Output
All three steps performed in background and gives the output.

Built for sales reps, agencies, and SaaS teams to prepare for calls faster.
